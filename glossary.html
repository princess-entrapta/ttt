Glossary

Action

The act of, for a character, proactively interact with the narrative by taking action. They can be of varied nature. See Actions And Reactions

Action point

Spendable resource a unit can use for doing various actions during combat. See Core Combat Mechanics

Character

Refers to a fictive person played by one of the players. Check their creation in Character Creation

For the purpose of combat, characters are represented as units having some combat characteristics. They are detailed in Core Combat Mechanics

Cover

A situational, defensive bonus that can origin from being partly obscured to attacker and/or partly covered by Terrain elevation. See Cover

Critical hit

Represent the fact for an attackre to reach at least twice the defense score with the roll. See Attack rules

Defense

Represent the minimum number to roll for an attacker to be able to successfuly land a full hit. See Core Combat Mechanics

Full hit

Represent the act, for an attacker, of beating the defense score with a roll. See Attack Rules

Game Master

Person who guide the game by doing the narrative, proposing player choices, and play enemies in combat.

Guidance

Bonus awarded to a character with a mastery helping at the roll of someone with less skill on an action or reaction roll. See Guidance

Health Points

Represent how much damage your character can take without dying. See Core Combat Mechanics

Line Of Sight

Unobstructed line from a unit to an other. What represent obstruction is detailed in Terrain features

Movement Points

Spendable resource a unit have to use for movement. See Core Combat Mechanics

Negative Trait

The quirk your character has, that make them act in a less optimal way when under Stress. See Non Combat Characteristics

Partial hit

Represent the fact for an attackre to reach half the defense score with the roll. See Attack rules

Players

Players refers to the persons that typically play a single character and roleplay through different game sessions. See Players

Reaction

The act, for a character, to react a certain narrative event. They can be of varied nature. See Non Combat Characteristics

Skills

The different actions and passive checks that can be rolled. They are Speech, Trading, Perception, Athletics, |Stealth, and Science, and can carry a bonus as described in Character Creation

Special Points

Spendable resource a character can spend for casting their drafted Abilities

Stress

Markers that indicate your Character is tense, that can have diverse impacting consequences. They are accumulated on Failures, and sometimes on Partial Success. See Stress

Tenacity Points

Represent a number of stunning effect a unit can take before being partially or completely stunned. See Core Combat Mechanics

Terrain elevation

The height of the terrain, determined by an integer. Can affect combat. See Terrain features
